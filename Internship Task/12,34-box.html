CSS Box Model Examples

Content Box (default box-sizing)

This box uses the traditional box model where width/height only apply to content.

Total width = width (350px)
Padding = 20px * 2
Border = 5px * 2
Border Box (box-sizing: border-box)

This box includes padding and border in the specified width/height.

Width = 350px (includes padding and border)
Content width = 350 - (20px * 2) - (5px * 2) = 300px
Percentage Width with em Units

This box uses percentage width and em units for padding/margin/border.

Width: 50% of parent
Padding: 2em (32px if base font-size is 16px)
Border: 1em (16px) double border
Max-Width with Viewport Units

This box won't exceed 600px width and has viewport-based margins.

Max-width: 600px
Margin: 10px 5% (5% of parent width)
Border only on top and bottom
Viewport Units

This box uses viewport units for width, height, padding, margin, and border.

Width: 80vw (80% of viewport width)
Height: 20vh (20% of viewport height)
Padding: 5vmin (5% of smaller viewport dimension)
